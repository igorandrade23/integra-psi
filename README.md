# Integra Psi

Website mobile first da chapa Integra Psi para o Centro Acadêmico Silvia Lane da Univali.

O projeto usa Next.js, TypeScript e Tailwind CSS, com uma organização modular inspirada em DDD e Clean Architecture. A ideia é manter o código legível para campanha, portfólio e referência pública.

## Stack

- Next.js com App Router
- TypeScript
- Tailwind CSS
- Lucide React
- Vercel

## Como rodar

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Estrutura

```txt
src/
  app/
  modules/
    proposals/
    team/
  shared/
    components/
    constants/
docs/
  PRD.md
```
